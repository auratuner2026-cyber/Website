Tuner

Tune anything.
Instantly.

Powered by the pYIN algorithm — the same pitch detection used in professional audio software. Accurate to ±0.1 cents, with real-time visual feedback that tells you exactly where you stand.

  • 20+ instrument presets — Guitar, Bass, Ukulele, Violin, 808 and more
  • Adjustable reference pitch — A4 from 432 Hz to 446 Hz
  • Solfège notation support — Do · Re · Mi alongside A · B · C
  • Haptic feedback when you hit the note perfectly

Key Finder

Hear the key.
Know the chord.

Real-time polyphonic analysis that identifies keys, scales, and diatonic chords as you play. Ideal for musicians jamming, transcribing, or learning by ear.

  • HPCP chromagram — 12 pitch classes visualized in real time
  • Krumhansl-Schmuckler key detection with confidence score
  • Diatonic chord analysis — see which chords belong to the key
  • System audio capture on Mac — analyze anything playing on your computer
